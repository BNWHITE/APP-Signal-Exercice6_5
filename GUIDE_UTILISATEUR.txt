3. LANCEMENT RAPIDE

   a) Ouvrir MATLAB
   b) Naviguer vers le dossier du projet :
      >> cd('/chemin/vers/APP-Signal-Exercice6_5')
   c) Executer le script principal :
      >> main
   d) Les figures s'affichent automatiquement :
      - Figure 1 : 4 sous-figures (signal original, decime sans filtre,
        filtre, decime avec filtre) en temps et frequence
      - Figure 2 : Reponse en frequence du filtre anti-repliement

4. LANCER LES TESTS UNITAIRES

   >> cd('/chemin/vers/APP-Signal-Exercice6_5')
   >> main          % pour ajouter les paths
   >> test_myDSPEstimation
   >> test_myKaiserFilter
   >> test_myDecimation

   Chaque fichier de test lance 10 cas et affiche PASS/FAIL.

5. ECOUTER LES SIGNAUX

   Dans exercice_6_5.m, decommentez les lignes "sound()" :
   - sound(x, Fe)    → signal original (44100 Hz)
   - sound(xl2, Fe2) → decime SANS filtre (repliement audible)
   - sound(xl4, Fe2) → decime AVEC filtre (qualite preservee)

6. CHANGER DE SIGNAL AUDIO

   Dans exercice_6_5.m, ligne ~30 :
   - [x, Fe] = audioread('GardeMontante.wav');   % par defaut
   - [x, Fe] = audioread('RavelSi.wav');         % decommenter pour tester

7. DESCRIPTION DES FONCTIONS

   myDSPEstimation(x, Fe, [])
     Entrees : signal x, frequence Fe
     Sorties : axe frequentiel f, module FFT xfm, DSP en dB dsp_db

   myKaiserFilter(Fe, Fpass, Fstop, Astop)
     Entrees : Fe=44100, Fpass=1800, Fstop=2000, Astop=40
     Sorties : coefficients h, ordre Nfilt
     Methode : fenetrage de Kaiser (sinc * fenetre), pas de toolbox

   myDecimation(x, Fe, Fe2, h)
     Entrees : signal x, Fe originale, Fe2 cible, filtre h (ou [] si aucun)
     Sorties : signal decime y, vecteur temps t2

8. RESUME DE L'EXERCICE

   Q1 : Chargement WAV stereo → mono, affichage temps + DSP
   Q2 : Decimation 44100 → 4000 Hz SANS filtre → repliement spectral
   Q3 : Conception filtre FIR passe-bas (Kaiser, Fpass=1800, Fstop=2000, 40 dB)
   Q4 : Filtrage du signal → composantes > 2 kHz attenuees de 40 dB
   Q5 : Decimation du signal filtre → pas de repliement, qualite preservee
